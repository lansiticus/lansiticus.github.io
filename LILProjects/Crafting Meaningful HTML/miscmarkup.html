Misc. HTML Markup

Controlling breaks

You can break the word Chlorofluorocarbons by using the <wbr> tag.

You can break and hyphenate the word Chloro­fluoro­carbons by using the &shy; entity.

Displaying Markup and Hiding Entities

To display HTML tags such as <heading> , use &lt; and &gt; like this: &lt;heading&gt;

To display entities such as &lt; , use &amp; like this &amp;lt;

Styling Code and Keypboard Commands
